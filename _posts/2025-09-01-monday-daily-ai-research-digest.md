### 1. A.S.E: A Repository-Level Benchmark for Evaluating Security in
  AI-Generated Code

[Read Paper](https://huggingface.co/papers/2508.18106)

A.S.E is a repository-level benchmark for evaluating the security of AI-generated code, highlighting challenges in secure coding and the limitations of LLMs in real-world scenarios.

### 2. A Survey of Scientific Large Language Models: From Data Foundations to
  Agent Frontiers

[Read Paper](https://huggingface.co/papers/2508.21148)

Sci-LLMs are evolving through a co-development with scientific data, addressing unique challenges like multimodal and domain-specific information, and are moving towards autonomous, closed-loop systems in scientific research.

### 3. R-4B: Incentivizing General-Purpose Auto-Thinking Capability in MLLMs
  via Bi-Mode Annealing and Reinforce Learning

[Read Paper](https://huggingface.co/papers/2508.21113)

R-4B, an auto-thinking multimodal large language model, uses bi-mode annealing and Bi-mode Policy Optimization to adaptively decide on problem-solving strategies, achieving state-of-the-art performance with lower computational cost.

### 4. EmbodiedOneVision: Interleaved Vision-Text-Action Pretraining for
  General Robot Control

[Read Paper](https://huggingface.co/papers/2508.21112)

EO-Robotics, comprising EO-1 model and EO-Data1.5M dataset, advances multimodal embodied reasoning and robot control through interleaved vision-text-action pre-training.

### 5. Droplet3D: Commonsense Priors from Videos Facilitate 3D Generation

[Read Paper](https://huggingface.co/papers/2508.20470)

Using video data to provide commonsense priors enhances 3D asset generation, enabling spatial consistency and semantic plausibility in 3D content creation.

### 6. Think in Games: Learning to Reason in Games via Reinforcement Learning
  with Large Language Models

[Read Paper](https://huggingface.co/papers/2508.21365)

Think in Games (TiG) framework enables large language models to develop procedural knowledge through interactive game environments, achieving competitive performance with reduced data and computational demands while providing transparent explanations.

### 7. Efficient Code Embeddings from Code Generation Models

[Read Paper](https://huggingface.co/papers/2508.21290)

Jina-code-embeddings uses an autoregressive backbone pre-trained on text and code to generate embeddings for code retrieval, question-answering, and similarity identification.

### 8. TalkVid: A Large-Scale Diversified Dataset for Audio-Driven Talking Head
  Synthesis

[Read Paper](https://huggingface.co/papers/2508.13618)

TalkVid, a large-scale, high-quality, and diverse dataset, improves audio-driven talking head synthesis by enhancing generalization across human diversity and revealing subgroup performance disparities.

### 9. TiKMiX: Take Data Influence into Dynamic Mixture for Language Model
  Pre-training

[Read Paper](https://huggingface.co/papers/2508.17677)

Dynamic adjustment of data mixture based on Group Influence metric improves language model performance by adapting to evolving learning preferences.

### 10. UItron: Foundational GUI Agent with Advanced Perception and Planning

[Read Paper](https://huggingface.co/papers/2508.21767)

UItron, an open-source foundational model for GUI agents, enhances visual understanding and task planning through advanced perception, grounding, and planning capabilities, achieving superior performance in Chinese app scenarios.
