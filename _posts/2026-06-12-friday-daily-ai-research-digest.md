### 1. EvoArena: Tracking Memory Evolution for Robust LLM Agents in Dynamic Environments

[Read Paper](https://huggingface.co/papers/2606.13681)

EvoArena benchmark and EvoMem memory paradigm address the challenge of dynamic environments in LLM agents by modeling progressive updates and structured memory evolution, showing improved performance on evolving tasks.

### 2. SpatialClaw: Rethinking Action Interface for Agentic Spatial Reasoning

[Read Paper](https://huggingface.co/papers/2606.13673)

SpatialClaw is a training-free framework that uses code as an action interface to enable flexible, stateful spatial reasoning in vision-language models, achieving superior performance across diverse 3D/4D spatial reasoning tasks.

### 3. MiniMax Sparse Attention

[Read Paper](https://huggingface.co/papers/2606.13392)

MiniMax Sparse Attention enables efficient processing of ultra-long contexts in large language models through blockwise sparsity and optimized GPU execution, achieving significant speedups while maintaining performance.

### 4. InterleaveThinker: Reinforcing Agentic Interleaved Generation

[Read Paper](https://huggingface.co/papers/2606.13679)

InterleaveThinker enables interleaved generation capabilities for image generators through a multi-agent pipeline with planner and critic agents, achieving performance comparable to state-of-the-art models while enhancing reasoning benchmarks.

### 5. FORT-Searcher: Synthesizing Shortcut-Resistant Search Tasks for Training Deep Search Agents

[Read Paper](https://huggingface.co/papers/2606.12087)

A framework for creating shortcut-resistant training data for deep search agents by identifying and mitigating four shortcut risks in data synthesis processes.

### 6. Robust-U1: Can MLLMs Self-Recover Corrupted Visual Content for Robust Understanding?

[Read Paper](https://huggingface.co/papers/2606.08063)

Robust-U1 enhances multimodal large language models' robustness against visual corruptions through self-recovery capabilities that improve both visual quality and reasoning performance.

### 7. MaxProof: Scaling Mathematical Proof with Generative-Verifier RL and Population-Level Test-Time Scaling

[Read Paper](https://huggingface.co/papers/2606.13473)

MaxProof is a test-time scaling framework that enhances mathematical proof generation by combining multiple proof-oriented capabilities and using population-level search with tournament selection to achieve competitive performance on high-level mathematical competitions.

### 8. WeaveBench: A Long-Horizon, Real-World Benchmark for Computer-Use Agents with Hybrid Interfaces

[Read Paper](https://huggingface.co/papers/2606.09426)

WeaveBench presents a comprehensive benchmark for evaluating computer-use agents across multiple interfaces, revealing significant challenges in long-horizon task orchestration and highlighting the limitations of traditional performance assessment methods.

### 9. LabVLA: Grounding Vision-Language-Action Models in Scientific Laboratories

[Read Paper](https://huggingface.co/papers/2606.13578)

LabVLA, a vision-language-action model trained with a two-stage approach combining action token pretraining and flow matching, demonstrates superior performance on laboratory automation tasks through simulated data generation and robot-specific learning.

### 10. HYDRA-X: Native Unified Multimodal Models with Holistic Visual Tokenizers

[Read Paper](https://huggingface.co/papers/2606.13289)

HYDRA-X presents a unified multimodal model that integrates image and video tokenization within a single Vision Transformer, addressing spatiotemporal reconstruction and semantic awareness through causal temporal attention and hierarchical compression.
