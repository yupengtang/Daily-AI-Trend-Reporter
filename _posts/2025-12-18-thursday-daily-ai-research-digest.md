### 1. Step-GUI Technical Report

[Read Paper](https://huggingface.co/papers/2512.15431)

A self-evolving training pipeline with the Calibrated Step Reward System and GUI-MCP protocol improve GUI automation efficiency, accuracy, and privacy in real-world scenarios.

### 2. Qwen-Image-Layered: Towards Inherent Editability via Layer Decomposition

[Read Paper](https://huggingface.co/papers/2512.15603)

A diffusion model for decomposing RGB images into semantically disentangled RGBA layers enables consistent image editing through independent manipulation of individual layers.

### 3. DEER: Draft with Diffusion, Verify with Autoregressive Models

[Read Paper](https://huggingface.co/papers/2512.15176)

Diffusion large language model drafters address limitations of autoregressive decoding in speculative frameworks through parallel processing and improved probabilistic modeling, achieving significant speedup in code generation tasks.

### 4. Universal Reasoning Model

[Read Paper](https://huggingface.co/papers/2512.14693)

The Universal Reasoning Model enhances Universal Transformers with short convolution and truncated backpropagation to improve reasoning performance on ARC-AGI tasks.

### 5. Fast and Accurate Causal Parallel Decoding using Jacobi Forcing

[Read Paper](https://huggingface.co/papers/2512.14681)

Parallel decoding for transformers is improved through a progressive distillation method that maintains causal inference properties while achieving significant speedup, combined with multi-block decoding that further reduces inference latency.

### 6. HyperVL: An Efficient and Dynamic Multimodal Large Language Model for Edge Devices

[Read Paper](https://huggingface.co/papers/2512.14052)

HyperVL, an efficient multimodal large language model for on-device inference, uses image tiling, Visual Resolution Compressor, and Dual Consistency Learning to reduce memory usage, latency, and power consumption while maintaining performance.

### 7. PuzzleCraft: Exploration-Aware Curriculum Learning for Puzzle-Based RLVR in VLMs

[Read Paper](https://huggingface.co/papers/2512.14944)

PuzzleCraft presents a supervision-free framework for scaling vision-centric RLVR using lightweight puzzle environments with built-in verification, incorporating an exploration-aware curriculum and a new consistency metric to improve reasoning robustness and downstream performance.

### 8. Can LLMs Guide Their Own Exploration? Gradient-Guided Reinforcement Learning for LLM Reasoning

[Read Paper](https://huggingface.co/papers/2512.15687)

G2RL is a gradient-guided reinforcement learning framework that improves large language model reasoning by aligning exploration with the model's own update geometry rather than using external heuristics.

### 9. MMSI-Video-Bench: A Holistic Benchmark for Video-Based Spatial Intelligence

[Read Paper](https://huggingface.co/papers/2512.10863)

MMSI-Video-Bench is a comprehensive benchmark for video-based spatial intelligence in MLLMs, revealing significant gaps between human and AI performance and highlighting challenges in geometric reasoning, motion grounding, and cross-video correspondence.

### 10. IC-Effect: Precise and Efficient Video Effects Editing via In-Context Learning

[Read Paper](https://huggingface.co/papers/2512.15635)

IC-Effect is a DiT-based framework for few-shot video VFX editing that synthesizes complex effects while maintaining spatial and temporal consistency through contextual learning and sparse tokenization.
