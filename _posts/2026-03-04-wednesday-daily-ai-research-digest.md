### 1. Utonia: Toward One Encoder for All Point Clouds

[Read Paper](https://huggingface.co/papers/2603.03283)

Utonia enables cross-domain point cloud representation learning through a unified self-supervised transformer encoder, enhancing perception and supporting embodied and multimodal reasoning tasks.

### 2. Beyond Language Modeling: An Exploration of Multimodal Pretraining

[Read Paper](https://huggingface.co/papers/2603.03276)

Controlled multimodal pretraining experiments reveal key insights about unified visual representations, data complementarity, world modeling emergence, and efficient scaling through mixture-of-experts architectures.

### 3. UniG2U-Bench: Do Unified Models Advance Multimodal Understanding?

[Read Paper](https://huggingface.co/papers/2603.03241)

Unified multimodal models show mixed performance in generation-to-understanding tasks, with specific subtasks benefiting from enhanced spatial and reasoning capabilities while overall performance lags behind specialized vision-language models.

### 4. Qwen3-Coder-Next Technical Report

[Read Paper](https://huggingface.co/papers/2603.00729)

Qwen3-Coder-Next is an 80-billion-parameter language model that activates only 3 billion parameters during inference, achieving strong coding capabilities through agentic training with verifiable task synthesis and reinforcement learning.

### 5. BeyondSWE: Can Current Code Agent Survive Beyond Single-Repo Bug Fixing?

[Read Paper](https://huggingface.co/papers/2603.03194)

Current code agent benchmarks fail to capture real-world complexity, prompting the creation of BeyondSWE to evaluate broader reasoning and knowledge scopes, alongside SearchSWE to study external knowledge integration in coding tasks.

### 6. Beyond Length Scaling: Synergizing Breadth and Depth for Generative Reward Models

[Read Paper](https://huggingface.co/papers/2603.01571)

Generative Reward Models can be improved by structuring Chain-of-Thought reasoning into breadth and depth components and optimizing them through supervised fine-tuning and reinforcement learning with verifiable rewards.

### 7. Kling-MotionControl Technical Report

[Read Paper](https://huggingface.co/papers/2603.03160)

Kling-MotionControl is a DiT-based framework for character animation that combines heterogeneous motion representations, adaptive identity-agnostic learning, and advanced acceleration techniques to achieve high-fidelity, expressive, and controllable video generation.

### 8. How Controllable Are Large Language Models? A Unified Evaluation across Behavioral Granularities

[Read Paper](https://huggingface.co/papers/2603.02578)

SteerEval is a hierarchical benchmark for evaluating large language model controllability across language features, sentiment, and personality domains with three specification levels.

### 9. Kiwi-Edit: Versatile Video Editing via Instruction and Reference Guidance

[Read Paper](https://huggingface.co/papers/2603.02175)

A scalable data generation pipeline creates high-fidelity video editing training data, and a unified architecture enables improved instruction-following and reference fidelity in controllable video editing.

### 10. Next Embedding Prediction Makes World Models Stronger

[Read Paper](https://huggingface.co/papers/2603.02765)

NE-Dreamer uses a temporal transformer to predict next-step encoder embeddings for model-based reinforcement learning without requiring decoders or auxiliary supervision.
