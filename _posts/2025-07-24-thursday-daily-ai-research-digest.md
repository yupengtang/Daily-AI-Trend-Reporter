### 1. Yume: An Interactive World Generation Model

[Read Paper](https://huggingface.co/papers/2507.17744)

A framework for generating and exploring interactive, high-fidelity video worlds from images using a Masked Video Diffusion Transformer, advanced sampling techniques, and model acceleration.

### 2. Pixels, Patterns, but No Poetry: To See The World like Humans

[Read Paper](https://huggingface.co/papers/2507.16863)

The Turing Eye Test evaluates MLLMs' perceptual capabilities on synthetic images, revealing that vision tower generalization is a key gap compared to human perception.

### 3. DesignLab: Designing Slides Through Iterative Detection and Correction

[Read Paper](https://huggingface.co/papers/2507.17202)

DesignLab, an iterative design tool using fine-tuned language models, outperforms existing methods by refining design drafts through continuous error detection and correction.

### 4. Can One Domain Help Others? A Data-Centric Study on Multi-Domain
  Reasoning via Reinforcement Learning

[Read Paper](https://huggingface.co/papers/2507.17512)

A study investigates multi-domain reasoning in Reinforcement Learning with Verifiable Rewards (RLVR) using the GRPO algorithm and Qwen-2.5-7B model family, exploring in-domain improvements, cross-domain generalization, and the impact of curriculum learning and reward design.

### 5. Ultra3D: Efficient and High-Fidelity 3D Generation with Part Attention

[Read Paper](https://huggingface.co/papers/2507.17745)

Ultra3D is an efficient 3D generation framework that uses VecSet for coarse layout and Part Attention for refined voxel features, achieving high-resolution generation with improved speed and quality.

### 6. RAVine: Reality-Aligned Evaluation for Agentic Search

[Read Paper](https://huggingface.co/papers/2507.16725)

RAVine is a new evaluation framework for agentic LLMs with search, focusing on realistic queries, accurate ground truth, and iterative process efficiency.

### 7. Re:Form -- Reducing Human Priors in Scalable Formal Software
  Verification with RL in LLMs: A Preliminary Study on Dafny

[Read Paper](https://huggingface.co/papers/2507.16331)

Formal language-based reasoning and automatic verification improve the reliability and scalability of Large Language Models in generating verifiable code.

### 8. Promptomatix: An Automatic Prompt Optimization Framework for Large
  Language Models

[Read Paper](https://huggingface.co/papers/2507.14241)

Promptomatix automates prompt optimization for Large Language Models, improving performance and efficiency across various tasks.

### 9. Elevating 3D Models: High-Quality Texture and Geometry Refinement from a
  Low-Quality Model

[Read Paper](https://huggingface.co/papers/2507.11465)

Elevate3D enhances low-quality 3D assets by iteratively refining both texture and geometry using specialized methods and monocular geometry predictors, achieving state-of-the-art quality.

### 10. PUSA V1.0: Surpassing Wan-I2V with $500 Training Cost by Vectorized
  Timestep Adaptation

[Read Paper](https://huggingface.co/papers/2507.16116)

Pusa, a vectorized timestep adaptation framework, enhances video diffusion models by enabling efficient and versatile temporal control, significantly reducing training costs and dataset sizes while improving performance and enabling zero-shot multi-task capabilities.
