### 1. HaluMem: Evaluating Hallucinations in Memory Systems of Agents

[Read Paper](https://huggingface.co/papers/2511.03506)

HaluMem, a benchmark for evaluating memory hallucinations in AI systems, identifies and analyzes hallucinations across memory extraction, updating, and question answering stages using large-scale human-AI interaction datasets.

### 2. IterResearch: Rethinking Long-Horizon Agents via Markovian State
  Reconstruction

[Read Paper](https://huggingface.co/papers/2511.07327)

IterResearch, an iterative deep-research paradigm, improves long-horizon reasoning by reformulating it as a Markov Decision Process with strategic workspace reconstruction and Efficiency-Aware Policy Optimization, achieving better performance and interaction scaling compared to existing agents.

### 3. DRIVE: Data Curation Best Practices for Reinforcement Learning with
  Verifiable Reward in Competitive Code Generation

[Read Paper](https://huggingface.co/papers/2511.06307)

The study presents a two-stage reinforcement learning approach for competitive-programming code generation, achieving state-of-the-art performance using Group Relative Policy Optimization and a hard-focus curriculum.

### 4. The Station: An Open-World Environment for AI-Driven Discovery

[Read Paper](https://huggingface.co/papers/2511.06309)

AI agents in the STATION environment achieve state-of-the-art performance across various benchmarks through autonomous scientific discovery and emergent behavior.

### 5. Robot Learning from a Physical World Model

[Read Paper](https://huggingface.co/papers/2511.07416)

PhysWorld integrates video generation and physical world modeling to enable accurate robotic manipulation from visual demonstrations without real robot data.

### 6. Generating an Image From 1,000 Words: Enhancing Text-to-Image With
  Structured Captions

[Read Paper](https://huggingface.co/papers/2511.06876)

A text-to-image model trained on long structured captions with DimFusion fusion mechanism and TaBR evaluation protocol achieves state-of-the-art prompt alignment and improved controllability.

### 7. Routing Manifold Alignment Improves Generalization of Mixture-of-Experts
  LLMs

[Read Paper](https://huggingface.co/papers/2511.07419)

Aligning routing weights with task embeddings in Sparse Mixture-of-Experts (MoE) models improves generalization and reduces performance gaps in large language models.

### 8. RedOne 2.0: Rethinking Domain-specific LLM Post-Training in Social
  Networking Services

[Read Paper](https://huggingface.co/papers/2511.07070)

RedOne 2.0, a social networking service-oriented LLM, uses a progressive, RL-prioritized post-training paradigm to achieve rapid and stable adaptation, delivering improvements over larger baselines with less data.

### 9. Reasoning with Confidence: Efficient Verification of LLM Reasoning Steps
  via Uncertainty Heads

[Read Paper](https://huggingface.co/papers/2511.06209)

Transformer-based uncertainty quantification heads improve step-level reasoning verification in LLMs by estimating uncertainty from internal states, offering a lightweight and scalable alternative to existing methods.

### 10. Teaching Pretrained Language Models to Think Deeper with Retrofitted
  Recurrence

[Read Paper](https://huggingface.co/papers/2511.07384)

Converting pretrained non-recurrent language models to depth-recurrent models improves performance at a given compute budget using a curriculum of recurrences.
