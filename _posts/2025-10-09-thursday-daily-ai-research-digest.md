### 1. Cache-to-Cache: Direct Semantic Communication Between Large Language
  Models

[Read Paper](https://huggingface.co/papers/2510.03215)

Cache-to-Cache (C2C) enables direct semantic communication between LLMs using neural network projections, improving accuracy and reducing latency compared to text-based communication.

### 2. Ming-UniVision: Joint Image Understanding and Generation with a Unified
  Continuous Tokenizer

[Read Paper](https://huggingface.co/papers/2510.06590)

MingTok, a continuous latent space visual tokenizer, unifies vision-language understanding and generation within an autoregressive framework, achieving state-of-the-art performance across both domains.

### 3. Lumina-DiMOO: An Omni Diffusion Large Language Model for Multi-Modal
  Generation and Understanding

[Read Paper](https://huggingface.co/papers/2510.06308)

Lumina-DiMOO, an open-source foundational model, uses fully discrete diffusion modeling for efficient multi-modal generation and understanding, outperforming existing models.

### 4. RLinf-VLA: A Unified and Efficient Framework for VLA+RL Training

[Read Paper](https://huggingface.co/papers/2510.06710)

RLinf-VLA is a unified framework for scalable reinforcement learning training of vision-language-action models, offering improved performance and generalization compared to supervised fine-tuning.

### 5. MATRIX: Mask Track Alignment for Interaction-aware Video Generation

[Read Paper](https://huggingface.co/papers/2510.07310)

MATRIX-11K dataset and MATRIX regularization enhance interaction fidelity and semantic alignment in video DiTs by aligning attention with multi-instance mask tracks.

### 6. SHANKS: Simultaneous Hearing and Thinking for Spoken Language Models

[Read Paper](https://huggingface.co/papers/2510.06917)

SHANKS, a general inference framework, enables spoken language models to generate unspoken reasoning while listening to user input, enhancing real-time interaction and task completion.

### 7. Vibe Checker: Aligning Code Evaluation with Human Preference

[Read Paper](https://huggingface.co/papers/2510.07315)

Vibe Checker evaluates LLMs by combining functional correctness and instruction following to better align with human coding preferences.

### 8. The Markovian Thinker

[Read Paper](https://huggingface.co/papers/2510.06557)

Markovian Thinking, implemented in Delethink, enables efficient and scalable reinforcement learning for long-chain-of-thought reasoning in LLMs by decoupling thinking length from context size, resulting in linear compute and constant memory usage.

### 9. Artificial Hippocampus Networks for Efficient Long-Context Modeling

[Read Paper](https://huggingface.co/papers/2510.07318)

A memory framework combining short-term and long-term memory in neural networks improves long-sequence modeling efficiency and performance.

### 10. Multi-Agent Tool-Integrated Policy Optimization

[Read Paper](https://huggingface.co/papers/2510.04678)

MATPO, a reinforcement learning method, optimizes tool-integrated multi-agent roles within a single LLM, improving performance and robustness over single-agent systems.
