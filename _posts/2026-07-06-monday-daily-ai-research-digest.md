### 1. The Mirage of Optimizing Training Policies: Monotonic Inference Policies as the Real Objective for LLM Reinforcement Learning

[Read Paper](https://huggingface.co/papers/2606.29526)

Training-inference mismatch in reinforcement learning for large language models leads to instability, which is addressed through a new policy optimization objective and framework that ensures consistent policy improvements between training and inference phases.

### 2. Embodied.cpp: A Portable Inference Runtime of Embodied AI Models on Heterogeneous Robots

[Read Paper](https://huggingface.co/papers/2607.02501)

Embodied.cpp is a portable C++ runtime that enables efficient deployment of vision-language-action and world-action models across heterogeneous edge devices through modular execution layers and optimized inference.

### 3. VLA-Corrector: Lightweight Detect-and-Correct Inference for Adaptive Action Horizon

[Read Paper](https://huggingface.co/papers/2607.01804)

VLA-Corrector addresses limitations of action chunking in vision-language-action models by introducing a lightweight latent-space vision monitor that enables adaptive corrective replanning, improving robustness in contact-rich manipulation tasks.

### 4. OrbitQuant: Data-Agnostic Quantization for Image and Video Diffusion Transformers

[Read Paper](https://huggingface.co/papers/2607.02461)

OrbitQuant enables efficient post-training quantization for diffusion transformers by using a normalized rotated basis that eliminates the need for recalibration across different timesteps and modalities.

### 5. DataComp-VLM: Improved Open Datasets for Vision-Language Models

[Read Paper](https://huggingface.co/papers/2606.28551)

DataComp for VLMs (DCVLM) establishes a comprehensive benchmark for evaluating data curation strategies in vision-language models, demonstrating that data mixing rather than filtering significantly improves model performance at scale.

### 6. Securing the AI Agent: A Unified Framework for Multi-Layer Agent Red Teaming

[Read Paper](https://huggingface.co/papers/2606.31227)

AI-Infra-Guard is an open-source framework that addresses AI infrastructure security through layered detection paradigms spanning infrastructure, protocol, agent behavior, and model layers.

### 7. Interpretation-Oriented Cloud Removal via Observation-Anchored Residual Flow with Geo-Contextual Alignment

[Read Paper](https://huggingface.co/papers/2607.02471)

Geo-Anchored Cloud Removal framework addresses semantic drift in cloud removal by combining physically grounded residual inversion with semantic manifold constraints from vision foundation models.

### 8. MultAttnAttrib: Training-Free Multimodal Attribution in Long Document Question Answering

[Read Paper](https://huggingface.co/papers/2607.01420)

MultAttnAttrib is a training-free multimodal attribution method that locates source evidence in documents using attention heads and calibrated thresholds, achieving superior accuracy and efficiency compared to existing approaches.

### 9. AGE: Adaptive-masking for Graph Embedding in Graph Retrieval-Augmented Generation

[Read Paper](https://huggingface.co/papers/2607.00052)

GraphRAG extends RAG by incorporating graph-structured data for LLMs, addressing latent feature misalignment through Adaptive-masking for Graph Embedding (AGE) that uses Transformer-based self-supervised learning with learnable node sampling.

### 10. AnyBokeh: Physics-Guided Any-to-Any Bokeh Editing with Optical Fingerprint Transfer

[Read Paper](https://huggingface.co/papers/2606.31959)

AnyBokeh is a physics-guided framework for bokeh editing that estimates source blur states and transfers optical characteristics between different focus and aperture settings without requiring all-in-focus reconstruction.
