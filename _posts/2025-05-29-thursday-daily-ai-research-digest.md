### 1. The Entropy Mechanism of Reinforcement Learning for Reasoning Language
  Models

[Read Paper](https://huggingface.co/papers/2505.22617)

Entropy dynamics in reinforcement learning with large language models are investigated to prevent policy entropy collapse and improve exploration.

### 2. SWE-rebench: An Automated Pipeline for Task Collection and
  Decontaminated Evaluation of Software Engineering Agents

[Read Paper](https://huggingface.co/papers/2505.20411)

A novel pipeline extracts real-world, interactive software engineering tasks from GitHub to create SWE-rebench, improving the evaluation of reinforcement learning models in SWE.

### 3. R2R: Efficiently Navigating Divergent Reasoning Paths with Small-Large
  Model Token Routing

[Read Paper](https://huggingface.co/papers/2505.21600)

Roads to Rome (R2R) selectively utilizes large language models for critical reasoning tasks to enhance efficiency and performance in lightweight models.

### 4. Skywork Open Reasoner 1 Technical Report

[Read Paper](https://huggingface.co/papers/2505.22312)

Skywork-OR1 is a reinforcement learning approach for long Chain-of-Thought models that improves accuracy over DeepSeek-R1 across various benchmarks by addressing entropy collapse.

### 5. Chain-of-Zoom: Extreme Super-Resolution via Scale Autoregression and
  Preference Alignment

[Read Paper](https://huggingface.co/papers/2505.18600)

Chain-of-Zoom (CoZ) enhances single-image super-resolution models by using an autoregressive chain of intermediate scale-states and multi-scale-aware prompts to achieve extreme magnifications with high quality.

### 6. Sherlock: Self-Correcting Reasoning in Vision-Language Models

[Read Paper](https://huggingface.co/papers/2505.22651)

Sherlock, a self-correction and self-improvement framework for reasoning vision-language models, enhances accuracy across benchmarks using limited annotated data.

### 7. Unsupervised Post-Training for Multi-Modal LLM Reasoning via GRPO

[Read Paper](https://huggingface.co/papers/2505.22453)

MM-UPT, a framework employing GRPO and self-rewarding, enhances multi-modal LLMs through unsupervised continual learning, showing performance improvements without manual annotations.

### 8. SageAttention2++: A More Efficient Implementation of SageAttention2

[Read Paper](https://huggingface.co/papers/2505.21136)

SageAttention2++ improves attention efficiency by using FP8 Matmul in FP16, achieving a 3.9x speedup over FlashAttention without losing accuracy.

### 9. RenderFormer: Transformer-based Neural Rendering of Triangle Meshes with
  Global Illumination

[Read Paper](https://huggingface.co/papers/2505.21925)

RenderFormer is a transformer-based neural rendering pipeline that renders images from triangle representations without per-scene training and with full global illumination effects.

### 10. Advancing Multimodal Reasoning via Reinforcement Learning with Cold
  Start

[Read Paper](https://huggingface.co/papers/2505.22334)

A two-stage approach combining supervised fine-tuning and reinforcement learning enhances multimodal reasoning in large language models, achieving state-of-the-art performance on benchmarks.
