### 1. Chain-of-Model Learning for Language Model

[Read Paper](https://huggingface.co/papers/2505.11820)

A novel Chain-of-Model framework introduces hierarchical hidden state chains in Transformers to improve scaling efficiency and inference flexibility for language models.

### 2. AdaptThink: Reasoning Models Can Learn When to Think

[Read Paper](https://huggingface.co/papers/2505.13417)

AdaptThink, an RL algorithm, enhances reasoning models' efficiency and performance by adaptively selecting the optimal thinking mode based on problem difficulty.

### 3. AdaCoT: Pareto-Optimal Adaptive Chain-of-Thought Triggering via
  Reinforcement Learning

[Read Paper](https://huggingface.co/papers/2505.11896)

AdaCoT, an adaptive reasoning framework using reinforcement learning, reduces unnecessary Chain-of-Thought generation by LLMs, cutting computational costs without sacrificing performance on complex tasks.

### 4. Thinkless: LLM Learns When to Think

[Read Paper](https://huggingface.co/papers/2505.13379)

Thinkless enables LLMs to adaptively choose between short and long reasoning by using control tokens, reducing computational inefficiencies on benchmarks.

### 5. Delta Attention: Fast and Accurate Sparse Attention Inference by Delta
  Correction

[Read Paper](https://huggingface.co/papers/2505.11254)

A novel procedure corrects distributional shifts in sparse attention, enhancing performance and reducing computational cost compared to full quadratic attention.

### 6. Scaling Computer-Use Grounding via User Interface Decomposition and
  Synthesis

[Read Paper](https://huggingface.co/papers/2505.13227)

A new benchmark OSWorld-G and dataset Jedi enhance GUI grounding by capturing complex interactions, leading to improved performance in computer use agents.

### 7. Model Merging in Pre-training of Large Language Models

[Read Paper](https://huggingface.co/papers/2505.12082)

Model merging during pre-training enhances large language models, improving performance and reducing costs across various architectures and learning rates.

### 8. Faster Video Diffusion with Trainable Sparse Attention

[Read Paper](https://huggingface.co/papers/2505.13389)

Trainable sparse attention (VSA) reduces computational cost in video diffusion transformers with minimal impact on performance, enabling efficient scaling of the models.

### 9. Hybrid 3D-4D Gaussian Splatting for Fast Dynamic Scene Representation

[Read Paper](https://huggingface.co/papers/2505.13215)

A new framework, 3D-4D Gaussian Splatting, improves computational efficiency in dynamic 3D scene reconstruction by adaptively representing static and dynamic regions, reducing parameter count and training time.

### 10. Through the Looking Glass: Common Sense Consistency Evaluation of Weird
  Images

[Read Paper](https://huggingface.co/papers/2505.07704)

A new method, Through the Looking Glass (TLG), uses Large Vision-Language Models and Transformer-based encoders to improve image common sense consistency assessment on the WHOOPS! and WEIRD datasets.
