### 1. Drag-and-Drop LLMs: Zero-Shot Prompt-to-Weights

[Read Paper](https://huggingface.co/papers/2506.16406)

Drag-and-Drop LLMs generate task-specific parameters through prompt-conditioned parameter generation, achieving significant efficiency gains and cross-domain generalization without per-task training.

### 2. Vision-Guided Chunking Is All You Need: Enhancing RAG with Multimodal
  Document Understanding

[Read Paper](https://huggingface.co/papers/2506.16035)

A novel multimodal document chunking approach using Large Multimodal Models (LMMs) enhances RAG performance by accurately processing complex PDF documents, including multi-page tables and embedded visuals.

### 3. PAROAttention: Pattern-Aware ReOrdering for Efficient Sparse and
  Quantized Attention in Visual Generation Models

[Read Paper](https://huggingface.co/papers/2506.16054)

PAROAttention reorganizes visual attention patterns to enable efficient sparsification and quantization, reducing memory and computational costs with minimal impact on performance.

### 4. Hunyuan-GameCraft: High-dynamic Interactive Game Video Generation with
  Hybrid History Condition

[Read Paper](https://huggingface.co/papers/2506.17201)

Hunyuan-GameCraft is a novel framework for high-dynamic interactive video generation in game environments that addresses limitations in dynamics, generality, and efficiency through unified input representation, hybrid history-conditioned training, and model distillation.

### 5. VIKI-R: Coordinating Embodied Multi-Agent Cooperation via Reinforcement
  Learning

[Read Paper](https://huggingface.co/papers/2506.09049)

VIKI-Bench and VIKI-R provide a benchmark and framework for evaluating and improving visual-driven cooperation among diverse embodied agents using vision-language models and reinforcement learning.

### 6. Hunyuan3D 2.5: Towards High-Fidelity 3D Assets Generation with Ultimate
  Details

[Read Paper](https://huggingface.co/papers/2506.16504)

Hunyuan3D 2.5, a suite of 3D diffusion models, advances shape and texture generation with a new LATTICE model and physical-based rendering in a multi-view architecture.

### 7. Machine Mental Imagery: Empower Multimodal Reasoning with Latent Visual
  Tokens

[Read Paper](https://huggingface.co/papers/2506.17218)

Mirage enhances vision-language models by integrating latent visual tokens into text decoding to improve multimodal reasoning without generating explicit images.

### 8. Optimizing Multilingual Text-To-Speech with Accents & Emotions

[Read Paper](https://huggingface.co/papers/2506.16310)

A new TTS architecture improves accent accuracy and emotion recognition for Hindi and Indian English by integrating phoneme alignment, culture-sensitive emotion embeddings, and dynamic accent code switching.

### 9. DreamCube: 3D Panorama Generation via Multi-plane Synchronization

[Read Paper](https://huggingface.co/papers/2506.17206)

Multi-plane synchronization extends 2D foundation models to 3D panorama generation, introducing DreamCube to achieve diverse appearances and accurate geometry.

### 10. Hunyuan3D 2.1: From Images to High-Fidelity 3D Assets with
  Production-Ready PBR Material

[Read Paper](https://huggingface.co/papers/2506.15442)

The tutorial provides a comprehensive guide on using Hunyuan3D 2.1 for generating high-resolution, textured 3D models, covering data preparation, model architecture, training, evaluation, and deployment.
