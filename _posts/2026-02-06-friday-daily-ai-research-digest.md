### 1. CAR-bench: Evaluating the Consistency and Limit-Awareness of LLM Agents under Real-World Uncertainty

[Read Paper](https://huggingface.co/papers/2601.22027)

Current LLM agent benchmarks fail to evaluate reliability in real-world scenarios with uncertain user inputs, prompting the creation of CAR-bench to test consistency, uncertainty management, and capability awareness in in-car assistant applications.

### 2. DFlash: Block Diffusion for Flash Speculative Decoding

[Read Paper](https://huggingface.co/papers/2602.06036)

DFlash is a speculative decoding framework that uses a lightweight block diffusion model for parallel token drafting, achieving significant speedup over existing autoregressive methods while maintaining high-quality outputs.

### 3. Spider-Sense: Intrinsic Risk Sensing for Efficient Agent Defense with Hierarchical Adaptive Screening

[Read Paper](https://huggingface.co/papers/2602.05386)

Spider-Sense framework provides intrinsic and selective agent security through event-driven defense with intrinsic risk sensing, achieving low attack success and false positive rates with minimal latency overhead.

### 4. MemSkill: Learning and Evolving Memory Skills for Self-Evolving Agents

[Read Paper](https://huggingface.co/papers/2602.02474)

MemSkill introduces a learnable and evolvable memory system for LLM agents that dynamically selects and refines memory operations through controller-executor-designer components.

### 5. Length-Unbiased Sequence Policy Optimization: Revealing and Controlling Response Length Variation in RLVR

[Read Paper](https://huggingface.co/papers/2602.05261)

Research analyzes RLVR algorithms' impact on response length in LLMs and VLMs, proposing LUSPO to eliminate length bias and improve reasoning performance.

### 6. Context Forcing: Consistent Autoregressive Video Generation with Long Context

[Read Paper](https://huggingface.co/papers/2602.06028)

Context Forcing addresses student-teacher mismatch in long video generation by using a long-context teacher to guide long-rollout students through a Slow-Fast Memory architecture that extends context length beyond 20 seconds.

### 7. Reinforced Attention Learning

[Read Paper](https://huggingface.co/papers/2602.04884)

Reinforced Attention Learning optimizes internal attention distributions in multimodal language models, improving information allocation and cross-modal alignment through policy-gradient methods.

### 8. Dr. Kernel: Reinforcement Learning Done Right for Triton Kernel Generations

[Read Paper](https://huggingface.co/papers/2602.05885)

Reinforcement learning approach for kernel generation addresses reward hacking and optimization issues through specialized environment and unbiased policy gradient methods, achieving competitive performance with state-of-the-art models.

### 9. Privileged Information Distillation for Language Models

[Read Paper](https://huggingface.co/papers/2602.04942)

Training methods that utilize privileged information for language model distillation in multi-turn environments outperform standard supervised fine-tuning followed by reinforcement learning approaches.

### 10. RISE-Video: Can Video Generators Decode Implicit World Rules?

[Read Paper](https://huggingface.co/papers/2602.05986)

RISE-Video presents a novel benchmark for evaluating text-image-to-video synthesis models based on cognitive reasoning rather than visual fidelity, using a multi-dimensional metric system and automated LMM-based evaluation.
