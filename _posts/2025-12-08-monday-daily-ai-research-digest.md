### 1. TwinFlow: Realizing One-step Generation on Large Models with Self-adversarial Flows

[Read Paper](https://huggingface.co/papers/2512.05150)

TwinFlow is a 1-step generative model framework that enhances inference efficiency without requiring fixed pretrained teacher models or standard adversarial networks, achieving high performance on text-to-image tasks and scaling efficiently.

### 2. EditThinker: Unlocking Iterative Reasoning for Any Image Editor

[Read Paper](https://huggingface.co/papers/2512.05965)

A deliberative editing framework with a reasoning engine improves instruction-following in image editing through iterative critique and refinement, significantly enhancing performance.

### 3. From Imitation to Discrimination: Toward A Generalized Curriculum Advantage Mechanism Enhancing Cross-Domain Reasoning Tasks

[Read Paper](https://huggingface.co/papers/2512.02580)

CAPO, a curriculum advantage policy optimization, enhances reinforcement learning for large language models by strategically introducing positive and negative advantage signals, improving reasoning capabilities and generalization.

### 4. RealGen: Photorealistic Text-to-Image Generation via Detector-Guided Rewards

[Read Paper](https://huggingface.co/papers/2512.00473)

RealGen is a photorealistic text-to-image framework that uses an LLM for prompt optimization and a diffusion model for image generation, enhanced by a Detector Reward mechanism and RealBench for automated evaluation.

### 5. EMMA: Efficient Multimodal Understanding, Generation, and Editing with a Unified Architecture

[Read Paper](https://huggingface.co/papers/2512.04810)

EMMA is an efficient unified architecture for multimodal tasks that uses autoencoders, channel-wise concatenation, shared-and-decoupled networks, and mixture-of-experts to achieve superior performance and efficiency.

### 6. SpaceControl: Introducing Test-Time Spatial Control to 3D Generative Modeling

[Read Paper](https://huggingface.co/papers/2512.05343)

SpaceControl enables explicit spatial control of 3D generation using various geometric inputs, outperforming existing methods in geometric faithfulness while maintaining visual quality.

### 7. PaCo-RL: Advancing Reinforcement Learning for Consistent Image Generation with Pairwise Reward Modeling

[Read Paper](https://huggingface.co/papers/2512.04784)

PaCo-RL combines reinforcement learning with a specialized consistency reward model and an efficient optimization strategy to improve consistent image generation.

### 8. SCAIL: Towards Studio-Grade Character Animation via In-Context Learning of 3D-Consistent Pose Representations

[Read Paper](https://huggingface.co/papers/2512.05905)

SCAIL framework improves character animation by using a novel 3D pose representation and a diffusion-transformer architecture with full-context pose injection, achieving studio-grade quality and realism.

### 9. Self-Improving VLM Judges Without Human Annotations

[Read Paper](https://huggingface.co/papers/2512.05145)

A framework for self-training a Vision-Language Model (VLM) judge using self-synthesized data improves judge accuracy on VL-RewardBench, surpassing larger models in several dimensions.

### 10. Entropy Ratio Clipping as a Soft Global Constraint for Stable Reinforcement Learning

[Read Paper](https://huggingface.co/papers/2512.05591)

Entropy Ratio Clipping (ERC) mechanism stabilizes policy updates in reinforcement learning by addressing global distributional shifts, improving performance over existing methods.
