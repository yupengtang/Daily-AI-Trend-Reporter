### 1. Matrix-Game: Interactive World Foundation Model

[Read Paper](https://huggingface.co/papers/2506.18701)

Matrix-Game, a controllable game world generation model trained in a two-stage process, outperforms existing models by producing high-quality, action-controllable, and physically consistent Minecraft world videos.

### 2. JarvisArt: Liberating Human Artistic Creativity via an Intelligent Photo
  Retouching Agent

[Read Paper](https://huggingface.co/papers/2506.17612)

JarvisArt, an MLLM-driven agent, achieves superior photo retouching by understanding user intent and coordinating multiple retouching tools in Lightroom, outperforming GPT-4o on a novel benchmark.

### 3. AnimaX: Animating the Inanimate in 3D with Joint Video-Pose Diffusion
  Models

[Read Paper](https://huggingface.co/papers/2506.19851)

AnimaX creates multi-skeleton 3D animations by blending video diffusion model priors with skeleton-based control, using joint video-pose diffusion and shared positional encodings.

### 4. Skywork-SWE: Unveiling Data Scaling Laws for Software Engineering in
  LLMs

[Read Paper](https://huggingface.co/papers/2506.19290)

An automated data-curation pipeline for software engineering improves large language model performance on SWE tasks, achieving state-of-the-art results with and without test-time scaling techniques.

### 5. Chain-of-Experts: Unlocking the Communication Power of
  Mixture-of-Experts Models

[Read Paper](https://huggingface.co/papers/2506.18945)

Chain-of-Experts (CoE) improves performance and memory efficiency in mixture-of-experts models by iteratively routing tokens through experts within each layer.

### 6. Unified Vision-Language-Action Model

[Read Paper](https://huggingface.co/papers/2506.19850)

UniVLA is a multimodal VLA model that autoregressively processes vision, language, and action as token sequences, incorporating world modeling for effective long-horizon policy learning and achieving state-of-the-art results across simulation and real-world benchmarks.

### 7. ScaleCap: Inference-Time Scalable Image Captioning via Dual-Modality
  Debiasing

[Read Paper](https://huggingface.co/papers/2506.19848)

ScaleCap enhances image captioning by iteratively enriching and calibrating captions using heuristic question answering and contrastive sentence rating, addressing multimodal and linguistic biases to improve accuracy, balance, and informativeness.

### 8. GRPO-CARE: Consistency-Aware Reinforcement Learning for Multimodal
  Reasoning

[Read Paper](https://huggingface.co/papers/2506.16141)

GRPO-CARE, a reinforcement learning framework optimizing for consistency and correctness, outperforms standard GRPO on a new video understanding benchmark, SEED-Bench-R1, improving both performance and logical coherence in multimodal large language models.

### 9. SWE-SQL: Illuminating LLM Pathways to Solve User SQL Issues in
  Real-World Applications

[Read Paper](https://huggingface.co/papers/2506.18951)

A new benchmark and training environment for debugging SQL issues using advanced open-source models significantly improves their performance compared to proprietary solutions.

### 10. Can Large Language Models Capture Human Annotator Disagreements?

[Read Paper](https://huggingface.co/papers/2506.19467)

LLMs struggle to predict human annotation disagreements, contrary to their performance in predicting majority labels, and RLVR-style reasoning exacerbates this issue.
