### 2. InteractComp: Evaluating Search Agents With Ambiguous Queries

[Read Paper](https://huggingface.co/papers/2510.24668)

InteractComp evaluates search agents' ability to recognize and resolve query ambiguity through interaction, revealing significant gaps in current models' capabilities.

### 3. AgentFold: Long-Horizon Web Agents with Proactive Context Management

[Read Paper](https://huggingface.co/papers/2510.24699)

AgentFold, a novel proactive context management paradigm, enhances long-horizon task performance through dynamic context folding, achieving superior results on benchmarks compared to larger models and proprietary agents.

### 4. RoboOmni: Proactive Robot Manipulation in Omni-modal Context

[Read Paper](https://huggingface.co/papers/2510.23763)

RoboOmni, a Perceiver-Thinker-Talker-Executor framework using end-to-end omni-modal LLMs, improves robotic manipulation by inferring user intentions from spoken dialogue, environmental sounds, and visual cues.

### 5. Game-TARS: Pretrained Foundation Models for Scalable Generalist
  Multimodal Game Agents

[Read Paper](https://huggingface.co/papers/2510.23691)

Game-TARS, a generalist game agent trained with a unified action space, achieves superior performance across various domains and benchmarks through large-scale pre-training and efficient reasoning strategies.

### 6. Uniform Discrete Diffusion with Metric Path for Video Generation

[Read Paper](https://huggingface.co/papers/2510.24717)

URSA, a discrete generative model, bridges the gap with continuous approaches in video generation by using iterative refinement, linearized metric paths, and resolution-dependent timestep shifting, achieving performance comparable to state-of-the-art continuous methods.

### 7. Agent Data Protocol: Unifying Datasets for Diverse, Effective
  Fine-tuning of LLM Agents

[Read Paper](https://huggingface.co/papers/2510.24702)

The agent data protocol (ADP) standardizes diverse agent training datasets, enabling improved performance across various tasks without domain-specific tuning.

### 8. From Spatial to Actions: Grounding Vision-Language-Action Model in
  Spatial Foundation Priors

[Read Paper](https://huggingface.co/papers/2510.17439)

FALCON enhances vision-language-action models by integrating rich 3D spatial tokens into the action head, improving spatial reasoning and modality transferability.

### 9. Group Relative Attention Guidance for Image Editing

[Read Paper](https://huggingface.co/papers/2510.24657)

Group Relative Attention Guidance enhances image editing quality by modulating token deltas in Diffusion-in-Transformer models, providing fine-grained control over editing intensity.

### 10. AgentFrontier: Expanding the Capability Frontier of LLM Agents with
  ZPD-Guided Data Synthesis

[Read Paper](https://huggingface.co/papers/2510.24695)

A ZPD-guided data synthesis approach enhances large language model capabilities by training them on frontier tasks with automated guidance, achieving state-of-the-art results.
