### 1. Your UnEmbedding Matrix is Secretly a Feature Lens for Text Embeddings

[Read Paper](https://huggingface.co/papers/2606.07502)

Text embeddings from large language models are enhanced by EmbedFilter, a linear transformation that reduces the influence of high-frequency tokens and improves semantic representations while enabling dimensionality reduction.

### 2. SoCRATES: Towards Reliable Automated Evaluation of Proactive LLM Mediation across Domains and Socio-cognitive Variations

[Read Paper](https://huggingface.co/papers/2606.05563)

SoCRATES presents a realistic multi-domain benchmark for evaluating proactive LLM mediators across various socio-cognitive adaptation axes, demonstrating that even top-performing models only resolve about one-third of the consensus gap in conflict resolution.

[Read Paper](https://huggingface.co/papers/2606.05806)

### 9. OpenSkill: Open-World Self-Evolution for LLM Agents

[Read Paper](https://huggingface.co/papers/2606.06741)

OpenSkill enables self-evolving agents to develop skills and verification signals from scratch using open-world resources without target-task supervision, achieving high automated performance across benchmarks.

### 10. SubtleMemory: A Benchmark for Fine-Grained Relational Memory Discrimination in Long-Horizon AI Agents

[Read Paper](https://huggingface.co/papers/2606.05761)

SubtleMemory benchmark evaluates AI agents' ability to handle complex relational memory structures that emerge during prolonged interactions, revealing limitations in current memory systems for preserving and utilizing nuanced memory relationships.
