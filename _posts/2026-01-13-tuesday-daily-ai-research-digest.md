### 4. MHLA: Restoring Expressivity of Linear Attention via Token-Level Multi-Head

[Read Paper](https://huggingface.co/papers/2601.07832)

Multi-Head Linear Attention addresses the performance degradation in linear attention by preserving representational diversity through head-wise token dimension computation, maintaining linear complexity while recovering softmax attention's expressive power across multiple domains.

### 5. X-Coder: Advancing Competitive Programming with Fully Synthetic Tasks, Solutions, and Tests

[Read Paper](https://huggingface.co/papers/2601.06953)

Code LLMs trained on fully synthetic data using a feature-based synthesis pipeline achieve superior performance on competitive programming benchmarks while reducing dependence on real-world coding datasets.

### 6. Lost in the Noise: How Reasoning Models Fail with Contextual Distractors

[Read Paper](https://huggingface.co/papers/2601.07226)

NoisyBench benchmark reveals significant performance degradation in state-of-the-art models when exposed to noisy contextual information, with agentic workflows amplifying errors and attention mechanisms disproportionately focusing on distractor tokens.

### 7. GlimpRouter: Efficient Collaborative Inference by Glimpsing One Token of Thoughts

[Read Paper](https://huggingface.co/papers/2601.05110)

Large reasoning models' inference latency can be reduced by routing reasoning steps to larger models based on the entropy of their first token, enabling efficient collaborative inference without additional training.

### 8. OS-Symphony: A Holistic Framework for Robust and Generalist Computer-Using Agent

[Read Paper](https://huggingface.co/papers/2601.07779)

OS-Symphony presents a comprehensive framework for computer-using agents that enhances robustness in long-horizon tasks through reflection-memory and multimodal search capabilities.

### 9. Beyond Hard Masks: Progressive Token Evolution for Diffusion Language Models

[Read Paper](https://huggingface.co/papers/2601.07351)

EvoToken-DLM introduces a diffusion-based language modeling approach that uses soft token distributions and continuous trajectory supervision to enable revisable decoding and outperforms existing baselines.

### 10. Controllable Memory Usage: Balancing Anchoring and Innovation in Long-Term Human-Agent Interaction

[Read Paper](https://huggingface.co/papers/2601.05107)
