### 1. LMEB: Long-horizon Memory Embedding Benchmark

[Read Paper](https://huggingface.co/papers/2603.12572)

A new benchmark evaluates embedding models' ability to handle long-horizon memory retrieval tasks, revealing that performance in traditional passage retrieval does not generalize to complex memory retrieval scenarios.

### 2. Multimodal OCR: Parse Anything from Documents

[Read Paper](https://huggingface.co/papers/2603.13032)

MOCR is a multimodal OCR approach that jointly parses text and graphics into unified representations, enabling structured document reconstruction and supporting end-to-end training with semantic relations between visual and textual components.

### 3. Can Vision-Language Models Solve the Shell Game?

[Read Paper](https://huggingface.co/papers/2603.08436)

Vision-Language Models exhibit poor performance on visual entity tracking due to reliance on static features; a proposed method using spatiotemporal grounded chain-of-thought achieves high accuracy by generating object trajectories as intermediate states.

### 4. Cheers: Decoupling Patch Details from Semantic Representations Enables Unified Multimodal Comprehension and Generation

[Read Paper](https://huggingface.co/papers/2603.12793)

Cheers is a unified multimodal model that decouples visual details from semantic representations using a vision tokenizer, LLM-based Transformer, and cascaded flow matching head to achieve efficient joint optimization for both visual understanding and generation tasks.

### 5. Video Streaming Thinking: VideoLLMs Can Watch and Think Simultaneously

[Read Paper](https://huggingface.co/papers/2603.12262)

Video Streaming Thinking (VST) introduces a novel streaming video understanding paradigm that enables real-time reasoning during video playback through causal streaming adaptation and multi-turn interaction optimization.

### 6. OmniForcing: Unleashing Real-time Joint Audio-Visual Generation

[Read Paper](https://huggingface.co/papers/2603.11647)

OmniForcing distills a dual-stream bidirectional diffusion model into a streaming autoregressive generator while addressing training instability and synchronization issues through asymmetric alignment and specialized token mechanisms.

### 7. daVinci-Env: Open SWE Environment Synthesis at Scale

[Read Paper](https://huggingface.co/papers/2603.13023)

OpenSWE presents the largest transparent framework for software engineering agent training, featuring 45,320 executable environments and achieving state-of-the-art performance on SWE-bench Verified.

### 8. Visual-ERM: Reward Modeling for Visual Equivalence

[Read Paper](https://huggingface.co/papers/2603.13224)

Visual-ERM, a multimodal generative reward model, provides fine-grained visual feedback for vision-to-code tasks, improving reinforcement learning performance and establishing a new benchmark for structured visual evaluation.

### 9. MM-CondChain: A Programmatically Verified Benchmark for Visually Grounded Deep Compositional Reasoning

[Read Paper](https://huggingface.co/papers/2603.12266)

MM-CondChain benchmark evaluates multimodal large language models on deep compositional visual reasoning through multi-layer conditional workflows with mechanically verifiable conditions.

### 10. Spend Less, Reason Better: Budget-Aware Value Tree Search for LLM Agents

[Read Paper](https://huggingface.co/papers/2603.12634)

A budget-aware value tree framework enables efficient multi-hop reasoning in language models by dynamically balancing exploration and exploitation based on remaining computational resources.
