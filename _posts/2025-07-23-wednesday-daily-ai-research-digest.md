### 1. Beyond Context Limits: Subconscious Threads for Long-Horizon Reasoning

[Read Paper](https://huggingface.co/papers/2507.16784)

The Thread Inference Model (TIM) and TIMRUN enable long-horizon reasoning in large language models by using reasoning trees and a rule-based subtask-pruning mechanism to manage working memory and GPU resources efficiently.

### 2. Step-Audio 2 Technical Report

[Read Paper](https://huggingface.co/papers/2507.16632)

Step-Audio~2, an end-to-end multi-modal large language model, integrates latent audio encoding and reinforcement learning to achieve state-of-the-art performance in ASR, audio understanding, and speech conversation, incorporating discrete audio token generation and retrieval-augmented generation.

### 3. MegaScience: Pushing the Frontiers of Post-Training Datasets for Science
  Reasoning

[Read Paper](https://huggingface.co/papers/2507.16812)

TextbookReasoning and MegaScience datasets, along with a comprehensive evaluation system, enhance scientific reasoning in AI by providing high-quality, verifiable data and outperforming existing models.

### 4. ThinkAct: Vision-Language-Action Reasoning via Reinforced Visual Latent
  Planning

[Read Paper](https://huggingface.co/papers/2507.16815)

ThinkAct, a dual-system framework, uses reinforced visual latent planning to enable few-shot adaptation, long-horizon planning, and self-correction in embodied AI tasks by bridging high-level reasoning with low-level action execution.

### 5. Upsample What Matters: Region-Adaptive Latent Sampling for Accelerated
  Diffusion Transformers

[Read Paper](https://huggingface.co/papers/2507.08422)

Region-Adaptive Latent Upsampling (RALU) accelerates diffusion transformer inference by performing mixed-resolution sampling, reducing computation while maintaining image quality.

### 6. Zebra-CoT: A Dataset for Interleaved Vision Language Reasoning

[Read Paper](https://huggingface.co/papers/2507.16746)

Zebra-CoT, a large-scale dataset with diverse visual and text reasoning tasks, improves multimodal model performance through fine-tuning, enhancing accuracy and visual chain of thought generation.

### 7. Semi-off-Policy Reinforcement Learning for Vision-Language Slow-thinking
  Reasoning

[Read Paper](https://huggingface.co/papers/2507.16814)

SOPHIA, a semi-off-policy reinforcement learning approach, enhances large vision-language models with slow-thinking reasoning, improving performance on multimodal reasoning tasks.

### 8. Experience is the Best Teacher: Grounding VLMs for Robotics through
  Self-Generated Memory

[Read Paper](https://huggingface.co/papers/2507.16713)

ExpTeach grounds vision-language models to physical robots through self-generated memory and retrieval-augmented generation, improving success rates and enabling intelligent object interactions.

### 9. RefCritic: Training Long Chain-of-Thought Critic Models with Refinement
  Feedback

[Read Paper](https://huggingface.co/papers/2507.15024)

RefCritic, a reinforcement learning-based critic module with dual rule-based rewards, enhances model critique and refinement capabilities, demonstrating superior performance across multiple benchmarks compared to supervised fine-tuning methods.

### 10. HOComp: Interaction-Aware Human-Object Composition

[Read Paper](https://huggingface.co/papers/2507.16813)

HOComp, a novel method using MLLMs and appearance preservation techniques, generates harmonious human-object interactions with consistent appearances in images.
