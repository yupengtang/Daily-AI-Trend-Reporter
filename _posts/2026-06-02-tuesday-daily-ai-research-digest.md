### 1. Crafter: A Multi-Agent Harness for Editable Scientific Figure Generation from Diverse Inputs

[Read Paper](https://huggingface.co/papers/2605.30611)

Automated systems for generating scientific figures face limitations in handling diverse figure types and conditions, prompting the development of multi-agent frameworks that generalize across different input scenarios and produce editable output formats.

### 2. On the Scaling of PEFT: Towards Million Personal Models of Trillion Parameters

[Read Paper](https://huggingface.co/papers/2606.02437)

Parameter-efficient fine-tuning can function as a compact substrate for persistent personal models by enabling small trainable adapters to store instance-specific behaviors on top of strong foundation models.

### 3. Domino: Decoupling Causal Modeling from Autoregressive Drafting in Speculative Decoding

[Read Paper](https://huggingface.co/papers/2605.29707)

Domino is a speculative decoding framework that improves LLM inference speed by decoupling causal dependency modeling from autoregressive drafting through a parallel backbone and lightweight causal refinement head, achieving significant speedups in both end-to-end execution and throughput.

### 4. A Matter of TASTE: Improving Coverage and Difficulty of Agent Benchmarks

[Read Paper](https://huggingface.co/papers/2605.28556)

Automated benchmark generation method creates challenging tasks with broader tool-use coverage by evolving tool sequences through adaptive contrastive n-gram modeling and iterative difficulty refinement.

### 5. Masking Stale Observations Helps Search Agents -- Until It Doesn't: A Regime Map and Its Mechanism

[Read Paper](https://huggingface.co/papers/2606.00408)

Observation masking in long-horizon search agents shows variable accuracy gains depending on the interaction between retriever capability and model capacity, following an asymmetric inverted-U pattern.

### 6. K-BrowseComp: A Web Browsing Agent Benchmark Grounded in Korean Contexts

[Read Paper](https://huggingface.co/papers/2606.02404)

Korean web-browsing agent benchmark K-BrowseComp evaluates frontier LLMs' capabilities with 400 problems, showing significant performance gaps compared to English benchmarks and highlighting the need for more robust Korean AI development.

### 7. Harness-1: Reinforcement Learning for Search Agents with State-Externalizing Harnesses

[Read Paper](https://huggingface.co/papers/2606.02373)

A 20B search agent trained with reinforcement learning within a stateful search framework demonstrates superior retrieval performance across multiple domains by separating semantic decision-making from environmental bookkeeping.

### 8. X-Stream: Exploring MLLMs as Multiplexers for Multi-Stream Understanding

[Read Paper](https://huggingface.co/papers/2606.02482)

X-Stream introduces the first benchmark for multi-stream streaming understanding, revealing significant limitations of current MLLMs in handling concurrent streams.

### 9. Draft-OPD: On-Policy Distillation for Speculative Draft Models

[Read Paper](https://huggingface.co/papers/2605.29343)

Speculative decoding uses a lightweight draft model to accelerate large language model inference, but supervised fine-tuning plateaus due to offline-to-inference mismatch, which is addressed through on-policy distillation with target-assisted rollouts and error replay.

### 10. NITP: Next Implicit Token Prediction for LLM Pre-training

[Read Paper](https://huggingface.co/papers/2605.24956)

Next Implicit Token Prediction enhances language model training by adding dense continuous supervision in representation space, improving generalization and performance across model sizes with minimal computational overhead.
