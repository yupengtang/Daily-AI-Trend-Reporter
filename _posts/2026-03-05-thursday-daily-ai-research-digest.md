### 1. Heterogeneous Agent Collaborative Reinforcement Learning

[Read Paper](https://huggingface.co/papers/2603.02604)

HACRL enables collaborative reinforcement learning where heterogeneous agents share verified rollouts during training to improve collectively while maintaining independent operation at inference time, with HACPO achieving superior performance through efficient sample utilization and cross-agent knowledge transfer.

### 2. Helios: Real Real-Time Long Video Generation Model

[Read Paper](https://huggingface.co/papers/2603.04379)

Helios is a 14 billion parameter autoregressive diffusion model for video generation that achieves real-time performance and high-quality long-video synthesis without conventional optimization techniques.

### 3. T2S-Bench & Structure-of-Thought: Benchmarking and Prompting Comprehensive Text-to-Structure Reasoning

[Read Paper](https://huggingface.co/papers/2603.03790)

Structure of Thought prompting technique enhances language model performance by guiding explicit intermediate text structuring across diverse tasks, while T2S-Bench benchmark evaluates and improves text-to-structure capabilities with comprehensive scientific domain coverage.

### 4. Proact-VL: A Proactive VideoLLM for Real-Time AI Companions

[Read Paper](https://huggingface.co/papers/2603.03447)

Proact-VL is a multimodal framework that enables real-time interactive AI companions for gaming scenarios with low-latency responses and strong video understanding capabilities.

### 5. MemSifter: Offloading LLM Memory Retrieval via Outcome-Driven Proxy Reasoning

[Read Paper](https://huggingface.co/papers/2603.03379)

MemSifter is a framework that uses a small proxy model to offload memory retrieval from large language models, employing reinforcement learning with task-performance rewards and training techniques like curriculum learning and model merging to improve long-term memory efficiency and accuracy.

### 6. ArtHOI: Articulated Human-Object Interaction Synthesis by 4D Reconstruction from Video Priors

[Read Paper](https://huggingface.co/papers/2603.04338)

ArtHOI synthesizes articulated human-object interactions by formulating 4D reconstruction from monocular video priors, using optical flow for part segmentation and a decoupled reconstruction pipeline to achieve physically plausible and geometrically consistent interactions.

### 7. Phi-4-reasoning-vision-15B Technical Report

[Read Paper](https://huggingface.co/papers/2603.03975)

A compact open-weight multimodal reasoning model is presented that achieves competitive performance through careful architecture design, high-quality data curation, and a hybrid approach combining direct answering with chain-of-thought reasoning.

### 8. Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory

[Read Paper](https://huggingface.co/papers/2603.04257)

A memory mechanism called Memex enables large language model agents to handle long-horizon tasks more effectively by maintaining compact context through structured summaries while storing full interaction details in an external database, allowing selective retrieval based on learned criteria.

### 9. Specificity-aware reinforcement learning for fine-grained open-world classification

[Read Paper](https://huggingface.co/papers/2603.03197)

A novel reinforcement learning framework called SpeciaRL is proposed to improve the specificity of large multimodal models in open-world fine-grained image classification while maintaining prediction correctness.

### 10. CubeComposer: Spatio-Temporal Autoregressive 4K 360° Video Generation from Perspective Video

[Read Paper](https://huggingface.co/papers/2603.04291)

CubeComposer is a spatio-temporal autoregressive diffusion model that generates high-resolution 360° panoramic videos by decomposing them into cubemap representations and using efficient autoregressive synthesis techniques.
