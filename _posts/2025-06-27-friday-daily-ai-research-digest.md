### 1. MMSearch-R1: Incentivizing LMMs to Search

[Read Paper](https://huggingface.co/papers/2506.20670)

MMSearch-R1, a reinforcement learning framework, enables large multimodal models to perform efficient, on-demand, multi-turn search in real-world environments, outperforming existing approaches.

### 2. Mind2Web 2: Evaluating Agentic Search with Agent-as-a-Judge

[Read Paper](https://huggingface.co/papers/2506.21506)

Mind2Web 2 benchmark evaluates agentic search systems with a suite of realistic, long-horizon tasks, introducing an Agent-as-a-Judge framework to assess accuracy and source attribution.

### 3. FaSTA^*: Fast-Slow Toolpath Agent with Subroutine Mining for Efficient
  Multi-turn Image Editing

[Read Paper](https://huggingface.co/papers/2506.20911)

A neurosymbolic agent combines language models for fast subtask planning with A$^*$ search for detailed toolpaths, creating a cost-efficient multi-turn image editing solution.

### 4. WorldVLA: Towards Autoregressive Action World Model

[Read Paper](https://huggingface.co/papers/2506.21539)

WorldVLA, an autoregressive action world model integrating vision-language-action (VLA) and world models, enhances performance through mutual understanding and generation, improving action prediction and sequence generation with an attention mask strategy.

### 5. MADrive: Memory-Augmented Driving Scene Modeling

[Read Paper](https://huggingface.co/papers/2506.21520)

MADrive enhances scene reconstruction for autonomous driving by integrating visually similar 3D car assets from an external memory bank to achieve photorealistic synthesis of altered scenarios.

### 6. Where to find Grokking in LLM Pretraining? Monitor
  Memorization-to-Generalization without Test

[Read Paper](https://huggingface.co/papers/2506.21551)

Grokking, or continued test performance improvement after training loss convergence, is observed during pretraining of a large language model, showcasing a memorization-to-generalization process.

### 7. Learning to Skip the Middle Layers of Transformers

[Read Paper](https://huggingface.co/papers/2506.21103)

A novel conditional computation architecture for Transformers dynamically skips middle layers based on input and a gating mechanism, but does not outperform dense baselines in reducing computational cost or improving validation performance.

### 8. Arch-Router: Aligning LLM Routing with Human Preferences

[Read Paper](https://huggingface.co/papers/2506.16655)

A preference-aligned routing framework using a compact 1.5B model effectively matches queries to user-defined domains and action types, outperforming proprietary models in subjective evaluation criteria.

### 9. SAM4D: Segment Anything in Camera and LiDAR Streams

[Read Paper](https://huggingface.co/papers/2506.21547)

SAM4D is a multi-modal and temporal foundation model for segmentation in autonomous driving using Unified Multi-modal Positional Encoding and Motion-aware Cross-modal Memory Attention, with a multi-modal automated data engine generating pseudo-labels.

### 10. PhysRig: Differentiable Physics-Based Skinning and Rigging Framework for
  Realistic Articulated Object Modeling

[Read Paper](https://huggingface.co/papers/2506.20936)

A physics-based skinning and rigging framework called PhysRig uses volumetric representation and continuum mechanics for more realistic and physically plausible animations.
