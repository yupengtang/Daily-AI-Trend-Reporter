### 1. Scaling Latent Reasoning via Looped Language Models

[Read Paper](https://huggingface.co/papers/2510.25741)

LoopLM, a family of pre-trained Looped Language Models, enhances reasoning by integrating iterative computation and entropy regularization during pre-training, achieving superior performance with better knowledge manipulation.

### 2. JanusCoder: Towards a Foundational Visual-Programmatic Interface for
  Code Intelligence

[Read Paper](https://huggingface.co/papers/2510.23538)

A unified multimodal code corpus and model enable high-quality code generation from both text and visual inputs, outperforming commercial models.

### 3. Video-Thinker: Sparking "Thinking with Videos" via Reinforcement
  Learning

[Read Paper](https://huggingface.co/papers/2510.23473)

Video-Thinker, a multimodal large language model, autonomously reasons with videos using intrinsic grounding and captioning capabilities, achieving state-of-the-art performance on various video reasoning benchmarks.

### 4. The Principles of Diffusion Models

[Read Paper](https://huggingface.co/papers/2510.21890)

Diffusion models are explored through variational, score-based, and flow-based perspectives, focusing on their mathematical foundations and applications in controllable generation and efficient sampling.

### 5. The Tool Decathlon: Benchmarking Language Agents for Diverse, Realistic,
  and Long-Horizon Task Execution

[Read Paper](https://huggingface.co/papers/2510.25726)

Toolathlon is a benchmark for language agents that evaluates their performance across diverse, realistic, and complex multi-step workflows involving multiple software applications and tools.

### 6. Ming-Flash-Omni: A Sparse, Unified Architecture for Multimodal
  Perception and Generation

[Read Paper](https://huggingface.co/papers/2510.24821)

Ming-Flash-Omni, an advanced Mixture-of-Experts model, enhances computational efficiency and multimodal intelligence, achieving state-of-the-art performance in speech recognition, image generation, and text-to-image generation.

### 7. Reasoning-Aware GRPO using Process Mining

[Read Paper](https://huggingface.co/papers/2510.25065)

PM4GRPO, a reasoning-aware Group Relative Policy Optimization, enhances policy models by incorporating process mining to align reasoning with a teacher model, outperforming existing methods.

### 8. VFXMaster: Unlocking Dynamic Visual Effect Generation via In-Context
  Learning

[Read Paper](https://huggingface.co/papers/2510.25772)

VFXMaster is a unified framework for generating VFX effects using in-context learning, enabling generalization to unseen effects and efficient adaptation from a single video.

### 9. RegionE: Adaptive Region-Aware Generation for Efficient Image Editing

[Read Paper](https://huggingface.co/papers/2510.25590)

RegionE is a region-aware framework that accelerates instruction-based image editing by differentiating between edited and unedited regions, optimizing the denoising process for each.

### 10. ODesign: A World Model for Biomolecular Interaction Design

[Read Paper](https://huggingface.co/papers/2510.22304)

ODesign is a generative world model that allows for the design of diverse biomolecular interactions across proteins, nucleic acids, and small molecules with fine-grained control.
