### 1. WebWatcher: Breaking New Frontier of Vision-Language Deep Research Agent

[Read Paper](https://huggingface.co/papers/2508.05748)

WebWatcher, a multimodal agent with enhanced visual-language reasoning, outperforms existing agents in complex visual and textual information retrieval tasks using synthetic trajectories and reinforcement learning.

### 2. Matrix-3D: Omnidirectional Explorable 3D World Generation

[Read Paper](https://huggingface.co/papers/2508.08086)

Matrix-3D generates wide-coverage 3D worlds from single images or text using panoramic video diffusion and reconstruction models.

### 3. Beyond Ten Turns: Unlocking Long-Horizon Agentic Search with Large-Scale
  Asynchronous RL

[Read Paper](https://huggingface.co/papers/2508.07976)

ASearcher is an open-source project that uses scalable asynchronous RL training to enhance search agents, achieving high performance on QA tasks with long-horizon search capabilities.

### 4. Complex Logical Instruction Generation

[Read Paper](https://huggingface.co/papers/2508.09125)

LogicIFGen and LogicIFEval assess the instruction-following capabilities of LLMs on complex, logic-rich instructions, revealing significant performance gaps.

### 5. VertexRegen: Mesh Generation with Continuous Level of Detail

[Read Paper](https://huggingface.co/papers/2508.09062)

VertexRegen generates meshes with continuous detail by reversing edge collapse through a generative model, offering anytime generation and flexibility in detail levels.

### 6. CharacterShot: Controllable and Consistent 4D Character Animation

[Read Paper](https://huggingface.co/papers/2508.07409)

CharacterShot is a 4D character animation framework that uses a DiT-based model and dual-attention module to generate consistent 3D animations from a single image and 2D pose sequence.

### 7. Time Is a Feature: Exploiting Temporal Dynamics in Diffusion Language
  Models

[Read Paper](https://huggingface.co/papers/2508.09138)

Two methods, Temporal Self-Consistency Voting and Temporal Consistency Reinforcement, improve diffusion large language models by leveraging temporal consistency in intermediate predictions.

### 8. OpenCUA: Open Foundations for Computer-Use Agents

[Read Paper](https://huggingface.co/papers/2508.09123)

OpenCUA is an open-source framework for vision-language models as computer-use agents, featuring an annotation infrastructure, a large-scale dataset, and a scalable pipeline that achieves state-of-the-art performance.

### 9. HierSearch: A Hierarchical Enterprise Deep Search Framework Integrating
  Local and Web Searches

[Read Paper](https://huggingface.co/papers/2508.08088)

HierSearch, a hierarchical agentic deep search framework using hierarchical RL, improves performance in multi-source retrieval tasks by coordinating local and Web search agents and refining knowledge.

### 10. Train Long, Think Short: Curriculum Learning for Efficient Reasoning

[Read Paper](https://huggingface.co/papers/2508.08940)

A curriculum learning strategy using Group Relative Policy Optimization (GRPO) enhances the reasoning abilities of large language models by progressively tightening token budgets, improving accuracy and token efficiency.
