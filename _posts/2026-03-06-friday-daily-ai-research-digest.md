### 1. SkillNet: Create, Evaluate, and Connect AI Skills

[Read Paper](https://huggingface.co/papers/2603.04448)

SkillNet introduces an open infrastructure for systematically accumulating and transferring AI skills through a unified ontology, significantly improving agent performance across multiple domains.

### 2. MOOSE-Star: Unlocking Tractable Training for Scientific Discovery by Breaking the Complexity Barrier

[Read Paper](https://huggingface.co/papers/2603.03756)

MOOSE-Star framework enables efficient training and inference for generative reasoning by addressing combinatorial complexity through decomposed subtasks, hierarchical search, and bounded composition.

### 3. DARE: Aligning LLM Agents with the R Statistical Ecosystem via Distribution-Aware Retrieval

[Read Paper](https://huggingface.co/papers/2603.04743)

A lightweight retrieval model called DARE incorporates data distribution information into function representations to improve R package retrieval, achieving superior performance over existing embedding models while enabling more reliable statistical analysis through an R-oriented LLM agent.

### 4. AgentVista: Evaluating Multimodal Agents in Ultra-Challenging Realistic Visual Scenarios

[Read Paper](https://huggingface.co/papers/2602.23166)

AgentVista presents a comprehensive benchmark for multimodal agents requiring long-horizon tool interactions across multiple modalities and complex real-world scenarios.

### 5. RoboPocket: Improve Robot Policies Instantly with Your Phone

[Read Paper](https://huggingface.co/papers/2603.05504)

RoboPocket enables efficient robot-free policy iteration using smartphone-based augmented reality visualization and asynchronous online fine-tuning to improve data and sample efficiency in imitation learning.

### 6. HiFi-Inpaint: Towards High-Fidelity Reference-Based Inpainting for Generating Detail-Preserving Human-Product Images

[Read Paper](https://huggingface.co/papers/2603.02210)

HiFi-Inpaint generates high-fidelity human-product images using shared enhancement attention and detail-aware loss with a new 40K-image dataset.

### 7. Large Multimodal Models as General In-Context Classifiers

[Read Paper](https://huggingface.co/papers/2602.23229)

Large Multimodal Models demonstrate superior performance in closed-world classification with in-context learning and excel in open-world scenarios when equipped with context refinement techniques.

### 8. MASQuant: Modality-Aware Smoothing Quantization for Multimodal Large Language Models

[Read Paper](https://huggingface.co/papers/2603.04800)

Post-training quantization for multimodal large language models addresses smoothing misalignment and cross-modal computational invariance through modality-aware smoothing and cross-modal compensation techniques.

### 9. Timer-S1: A Billion-Scale Time Series Foundation Model with Serial Scaling

[Read Paper](https://huggingface.co/papers/2603.04791)

Timer-S1 is a scalable Mixture-of-Experts time series model with 8.3B parameters that uses serial scaling and novel TimeMoE blocks to improve long-term forecasting accuracy.
