### 1. T-pro 2.0: An Efficient Russian Hybrid-Reasoning Model and Playground

[Read Paper](https://huggingface.co/papers/2512.10430)

T-pro 2.0 is an open-weight Russian LLM for hybrid reasoning and efficient inference, using a Cyrillic-dense tokenizer and EAGLE speculative-decoding pipeline.

### 2. Are We Ready for RL in Text-to-3D Generation? A Progressive Investigation

[Read Paper](https://huggingface.co/papers/2512.10949)

This study investigates reinforcement learning for text-to-3D generation, focusing on reward designs, RL algorithms, benchmarking, and hierarchical optimization, introducing AR3D-R1 as the first RL-enhanced model for 3D generation.

### 3. Long-horizon Reasoning Agent for Olympiad-Level Mathematical Problem Solving

[Read Paper](https://huggingface.co/papers/2512.10739)

OPV, an iterative active learning framework with Rejection Fine-Tuning, enhances verification of long reasoning chains in large language models, achieving state-of-the-art results and improving accuracy in collaborative tasks.

### 4. BEAVER: An Efficient Deterministic LLM Verifier

[Read Paper](https://huggingface.co/papers/2512.05439)

BEAVER is a framework that provides deterministic and sound probability bounds for verifying constraints in large language models, achieving tighter bounds and identifying more high-risk instances than baseline methods.

### 5. OPV: Outcome-based Process Verifier for Efficient Long Chain-of-Thought Verification

[Read Paper](https://huggingface.co/papers/2512.10756)

The Outcome-based Process Verifier (OPV) improves the verification of complex reasoning chains in large language models by combining outcome-based and process-based verification with iterative active learning and Rejection Fine-Tuning, achieving state-of-the-art performance on various benchmarks.

### 6. MoCapAnything: Unified 3D Motion Capture for Arbitrary Skeletons from Monocular Videos

[Read Paper](https://huggingface.co/papers/2512.10881)

MoCapAnything is a reference-guided framework that reconstructs rotation-based animations from monocular video for arbitrary rigged 3D assets, enabling cross-species retargeting and scalable 3D motion capture.

### 7. Achieving Olympia-Level Geometry Large Language Model Agent via Complexity Boosting Reinforcement Learning

[Read Paper](https://huggingface.co/papers/2512.10534)

InternGeometry, an LLM agent, surpasses human performance on IMO geometry problems using a heuristic-driven approach with iterative proposition verification and a dynamic memory mechanism, significantly outperforming AlphaGeometry 2 with limited training data.

### 8. Stronger Normalization-Free Transformers

[Read Paper](https://huggingface.co/papers/2512.10938)

Derf, a novel point-wise normalization function, outperforms existing alternatives across various domains, enhancing generalization without increased fitting capacity.

### 9. Thinking with Images via Self-Calling Agent

[Read Paper](https://huggingface.co/papers/2512.08511)

sCoT, a language-only CoT paradigm with self-calling subagents, enhances visual reasoning performance and efficiency through group-relative policy optimization.

### 10. Evaluating Gemini Robotics Policies in a Veo World Simulator

[Read Paper](https://huggingface.co/papers/2512.10675)

A generative evaluation system using a frontier video model (Veo) enables comprehensive policy evaluation in robotics, including nominal performance, out-of-distribution generalization, and safety checks.
