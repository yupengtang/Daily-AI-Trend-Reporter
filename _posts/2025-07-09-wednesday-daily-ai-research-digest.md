### 1. SingLoRA: Low Rank Adaptation Using a Single Matrix

[Read Paper](https://huggingface.co/papers/2507.05566)

SingLoRA, a reformulated low-rank adaptation method, enhances parameter-efficient fine-tuning by learning a single low-rank matrix and its transpose, ensuring stable optimization and reducing parameter count.

### 2. A Survey on Latent Reasoning

[Read Paper](https://huggingface.co/papers/2507.06203)

Latent reasoning in Large Language Models (LLMs) performs multi-step inference in continuous hidden states, enhancing reasoning capabilities without token-level supervision, and includes methodologies like activation-based recurrence and infinite-depth reasoning via masked diffusion models.

### 3. Agent KB: Leveraging Cross-Domain Experience for Agentic Problem Solving

[Read Paper](https://huggingface.co/papers/2507.06229)

Agent KB, a hierarchical experience framework, enhances problem-solving success rates across different agents by enabling cross-agent knowledge transfer through a Reason-Retrieve-Refine pipeline.

### 4. OmniPart: Part-Aware 3D Generation with Semantic Decoupling and
  Structural Cohesion

[Read Paper](https://huggingface.co/papers/2507.06165)

OmniPart generates part-aware 3D objects with high semantic decoupling and structural cohesion using an autoregressive structure planning module and a spatially-conditioned rectified flow model.

### 5. How to Train Your LLM Web Agent: A Statistical Diagnosis

[Read Paper](https://huggingface.co/papers/2507.04103)

A study on compute allocation for post-training LLM-based web agents finds that combining supervised fine-tuning with on-policy reinforcement learning improves performance and reduces computational costs compared to using either method alone.

### 6. StreamVLN: Streaming Vision-and-Language Navigation via SlowFast Context
  Modeling

[Read Paper](https://huggingface.co/papers/2507.05240)

StreamVLN, a streaming VLN framework, uses a hybrid slow-fast context modeling strategy to achieve state-of-the-art performance with low latency and efficient resource usage.

### 7. CriticLean: Critic-Guided Reinforcement Learning for Mathematical
  Formalization

[Read Paper](https://huggingface.co/papers/2507.06181)

CriticLean, a critic-guided reinforcement learning framework, enhances the evaluation of formalizations in automated theorem proving by actively learning and distinguishing semantically correct formalizations.

### 8. RLVER: Reinforcement Learning with Verifiable Emotion Rewards for
  Empathetic Agents

[Read Paper](https://huggingface.co/papers/2507.03112)

RLVER, an end-to-end reinforcement learning framework using verifiable emotion rewards, enhances emotional intelligence in LLMs without sacrificing cognitive abilities.

### 9. GTA1: GUI Test-time Scaling Agent

[Read Paper](https://huggingface.co/papers/2507.05791)

A GUI Test-time Scaling Agent addresses task planning ambiguity and visual grounding accuracy in GUI interactions using reinforcement learning and test-time scaling.

### 10. MedGen: Unlocking Medical Video Generation by Scaling
  Granularly-annotated Medical Videos

[Read Paper](https://huggingface.co/papers/2507.05675)

MedGen, a model trained on MedVideoCap-55K, achieves top performance in medical video generation, balancing visual quality and medical accuracy.
